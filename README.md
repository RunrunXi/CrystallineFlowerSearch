# CrystallineFlowerSearch

![License](https://img.shields.io/badge/license-MIT-blue.svg)

为晶化花添加了搜索框，可以方便的搜索附魔。

> A search box has been added to the Crystalline Flower, allowing for convenient searching of enchantments.

## 📖 背景 | Background

在蜜蜂领域（The Bumblezone）模组中，**晶化花（Crystalline Flower）** 是一个重要的功能性方块。玩家可以通过消耗经验值，在晶化花中**主动选择任意附魔**，将其赋予到书本上，从而制作出自己想要的附魔书。

然而，随着模组越装越多，游戏中的附魔种类可能多达数十甚至上百种。晶化花的附魔选择界面将所有附魔杂乱地排列在一起，玩家想要找到某个特定附魔时，只能一个个慢慢翻找，体验十分不便。

**CrystallineFlowerSearch** 正是为了解决这个问题而诞生的。

> In The Bumblezone mod, the **Crystalline Flower** is an important functional block. Players can spend experience points to **actively select any enchantment** and apply it to a book, crafting the enchanted book they desire.
>
> However, as more mods are installed, the number of enchantments in the game can reach dozens or even hundreds. The Crystalline Flower's enchantment selection GUI displays all enchantments in a messy list, forcing players to scroll through them one by one when looking for a specific enchantment — a very inconvenient experience.
>
> **CrystallineFlowerSearch** was created to solve this problem.

---

## ✨ 功能 | Features

- 在晶化花的附魔选择界面中添加了一个**搜索框**
- 输入关键词即可**实时过滤**附魔列表，快速定位目标附魔
- 支持**中文**和**英文**搜索（根据游戏语言自动适配）
- 轻量实现，不影响晶化花的原有功能

> - Adds a **search box** to the Crystalline Flower's enchantment selection GUI.
> - **Real-time filtering** — type a keyword and the list updates instantly.
> - Supports both **Chinese** and **English** search (automatically adapts to game language).
> - Lightweight implementation, does not interfere with the original functionality.
